## ⚡ Competitive Programming Concept: Longest Common Subsequence

**Category:** DP
**TL;DR:** Mencari subsequence terpanjang yang sama antara dua string dengan pendekatan bottom-up Dynamic Programming.

Konsep Longest Common Subsequence (LCS) adalah masalah klasik dalam pemrograman dinamis yang bertujuan untuk menemukan subsequence terpanjang yang ada di antara dua string, di mana urutan karakter harus dipertahankan tetapi tidak harus berurutan (kontigu). Sebagai contoh, jika string pertama adalah "ABCBDAB" dan string kedua adalah "BDCABA", maka salah satu LCS-nya adalah "BCBA" dengan panjang 4. LCS banyak diterapkan dalam bidang seperti komparasi file (diff utility) dan bioinformatika.

Untuk menyelesaikan masalah LCS, kita dapat menggunakan pendekatan Dynamic Programming. Kita mendefinisikan sebuah tabel `dp[i][j]` yang menyimpan panjang LCS dari prefix string pertama `s1` dengan panjang `i` (yaitu `s1[0...i-1]`) dan prefix string kedua `s2` dengan panjang `j` (yaitu `s2[0...j-1]`). Base case adalah `dp[0][j] = 0` dan `dp[i][0] = 0`, karena LCS dengan string kosong selalu 0.

Rekurensi DP-nya bergantung pada perbandingan karakter `s1[i-1]` dan `s2[j-1]`. Jika kedua karakter tersebut sama, maka kita bisa memperpanjang LCS yang ditemukan dari prefix sebelumnya, sehingga `dp[i][j] = 1 + dp[i-1][j-1]`. Jika karakter berbeda, maka kita harus memilih yang terbaik dari dua kemungkinan: LCS dari `s1[0...i-2]` dan `s2[0...j-1]` (yaitu `dp[i-1][j]`), atau LCS dari `s1[0...i-1]` dan `s2[0...j-2]` (yaitu `dp[i][j-1]`). Jadi, `dp[i][j] = max(dp[i-1][j], dp[i][j-1])`. Setelah tabel terisi penuh, `dp[n][m]` (di mana `n` dan `m` adalah panjang string `s1` dan `s2`) akan berisi panjang LCS total.

Kompleksitas waktu untuk algoritma ini adalah O(N*M), di mana N adalah panjang string pertama dan M adalah panjang string kedua, karena kita mengisi setiap sel dalam tabel N x M satu kali. Kompleksitas ruangnya juga O(N*M) untuk menyimpan tabel DP, meskipun bisa dioptimasi menjadi O(min(N, M)) dengan hanya menyimpan dua baris atau kolom terakhir.

```
int lcs(const std::string& s1, const std::string& s2) {
    int n = s1.length();
    int m = s2.length();
    std::vector<std::vector<int>> dp(n + 1, std::vector<int>(m + 1, 0));
    for (int i = 1; i <= n; ++i) {
        for (int j = 1; j <= m; ++j) {
            if (s1[i - 1] == s2[j - 1]) {
                dp[i][j] = 1 + dp[i - 1][j - 1];
            } else {
                dp[i][j] = std::max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
    }
    return dp[n][m];
}
```

💡 **Fun Fact:** Algoritma LCS memiliki peran penting dalam bioinformatika untuk mengukur kesamaan antara dua urutan DNA atau protein, membantu dalam identifikasi gen dan analisis filogenetik.
