## ☕ Java Concept: ArrayList: Dynamic Array di Java

**Category:** Collections
**TL;DR:** ArrayList adalah implementasi array dinamis yang memungkinkan Anda menyimpan dan mengelola koleksi objek yang ukurannya dapat berubah secara otomatis.

Dalam pemrograman, kita seringkali perlu menyimpan koleksi data. Array standar di Java memiliki ukuran yang tetap setelah dibuat, yang bisa menjadi batasan ketika kita tidak tahu berapa banyak elemen yang akan kita butuhkan. Di sinilah ArrayList datang sebagai solusi praktis. ArrayList adalah implementasi array dinamis yang memungkinkan Anda menyimpan dan mengelola koleksi objek yang ukurannya dapat berubah secara otomatis saat elemen ditambahkan atau dihapus. Ini memudahkan pengelolaan data tanpa harus pusing memikirkan kapasitas awal.

Untuk mulai menggunakan ArrayList, Anda cukup membuatnya dengan menentukan tipe data objek yang akan disimpannya, misalnya `ArrayList<String>`. Anda kemudian bisa menambahkan elemen menggunakan metode `add()`, mengakses elemen dengan `get(index)`, atau mengetahui jumlah elemen saat ini dengan `size()`. Kemampuan untuk tumbuh dan menyusut secara otomatis inilah yang membuat ArrayList sangat populer dan sering digunakan dalam berbagai aplikasi.

Secara internal, ArrayList sebenarnya menggunakan array biasa sebagai dasarnya. Ketika ArrayList mendeteksi bahwa array internalnya sudah penuh saat Anda mencoba menambahkan elemen baru, ia akan membuat array baru yang lebih besar, menyalin semua elemen lama ke array baru tersebut, dan membuang array lama. Proses ini memang membutuhkan sedikit overhead, tetapi umumnya sangat efisien dan transparan bagi pengembang. Penggunaan generics (`<String>`, `<Integer>`) juga memastikan tipe data yang aman dan menghindari kesalahan saat runtime.

ArrayList adalah pilihan yang sangat baik ketika Anda membutuhkan daftar objek yang fleksibel, di mana operasi penambahan di akhir, penghapusan, dan akses berdasarkan indeks sering dilakukan. Meskipun ada struktur data koleksi lain seperti `LinkedList` atau `HashSet`, ArrayList adalah titik awal yang sempurna untuk memahami konsep koleksi dinamis di Java dan merupakan salah satu kelas yang paling sering Anda temui dalam pengembangan Java sehari-hari.

```
import java.util.ArrayList;

public class MahasiswaList {
    public static void main(String[] args) {
        ArrayList<String> daftarMahasiswa = new ArrayList<>(); // Membuat ArrayList
        daftarMahasiswa.add("Alice"); // Menambah elemen
        daftarMahasiswa.add("Bob");
        System.out.println("Jumlah mahasiswa: " + daftarMahasiswa.size()); // Mendapatkan ukuran
        System.out.println("Mahasiswa pertama: " + daftarMahasiswa.get(0)); // Mengakses elemen
        daftarMahasiswa.add("Charlie");
        System.out.println("Daftar lengkap: " + daftarMahasiswa); // Mencetak seluruh daftar
    }
}
```

💡 **Fun Fact:** ArrayList adalah bagian dari Java Collections Framework yang diperkenalkan di Java 1.2, merevolusi cara pengembang mengelola grup objek dibandingkan array klasik yang berukuran tetap.
