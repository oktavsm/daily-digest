## ⚡ Competitive Programming Concept: Segment Tree

**Category:** Data Structure
**TL;DR:** Segment Tree adalah struktur data pohon yang efisien untuk melakukan query (pertanyaan) dan update (pembaruan) pada range (interval) data dalam waktu logaritmik.

Segment Tree adalah struktur data pohon biner yang digunakan untuk menyimpan informasi tentang interval atau segmen. Setiap node dalam pohon merepresentasikan sebuah interval dari array asli, dengan node akar merepresentasikan seluruh array, dan node-node anak membagi interval tersebut menjadi dua sub-interval yang lebih kecil. Struktur ini memungkinkan operasi seperti "mencari jumlah/nilai minimum/maksimum dalam suatu range" atau "mengubah nilai pada indeks tertentu" dilakukan secara sangat cepat.

Untuk melakukan operasi query (misalnya, mencari jumlah elemen dalam range [L, R]), Segment Tree bekerja dengan memecah range query menjadi beberapa interval yang sudah direpresentasikan oleh node-node dalam pohon. Karena setiap node menyimpan agregasi informasi (seperti jumlah atau nilai minimum) untuk intervalnya, kita tidak perlu memindai setiap elemen dalam range. Proses ini bersifat rekursif dan hanya melibatkan jalur-jalur tertentu dalam pohon.

Demikian pula, untuk operasi update (misalnya, mengubah nilai elemen pada indeks I), kita hanya perlu memperbarui nilai pada node daun yang sesuai dengan indeks I, lalu memperbarui nilai pada semua node 'ancestor' (nenek moyang) dari node daun tersebut hingga ke akar. Karena Segment Tree adalah pohon biner yang seimbang, tinggi pohon hanya logaritmik terhadap jumlah elemen, sehingga hanya sedikit node yang perlu diubah.

Pembangunan (build) Segment Tree membutuhkan waktu O(N), di mana N adalah ukuran array asli. Setelah dibangun, setiap operasi query dan update dapat diselesaikan dalam kompleksitas waktu O(log N). Ini membuatnya sangat efisien untuk masalah di mana banyak query dan update perlu dilakukan pada array yang sama.

```
const int N = 1e5;
int tree[4*N]; // Array untuk menyimpan tree, ukuran 4N untuk keamanan
int arr[N];    // Array asli

void update(int node, int start, int end, int idx, int val) {
    if(start == end) { arr[idx] = val; tree[node] = val; return; }
    int mid = (start + end) / 2;
    if(start <= idx && idx <= mid) update(2*node, start, mid, idx, val);
    else update(2*node+1, mid+1, end, idx, val);
    tree[node] = tree[2*node] + tree[2*node+1]; // Contoh: sum
}
```

💡 **Fun Fact:** Segment Tree sering digunakan bersamaan dengan teknik 'Lazy Propagation' untuk mengoptimalkan update pada range yang besar. Teknik ini menunda penerapan update hingga benar-benar diperlukan, menjaga efisiensi O(log N) bahkan untuk update range.
