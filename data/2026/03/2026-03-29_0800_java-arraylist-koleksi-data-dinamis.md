## ☕ Java Concept: Java ArrayList: Koleksi Data Dinamis

**Category:** Collections
**TL;DR:** ArrayList adalah struktur data dinamis di Java yang memungkinkan kita menyimpan dan mengelola kumpulan objek dengan mudah tanpa perlu tahu ukurannya di awal.

Dalam pemrograman, kita seringkali perlu menyimpan lebih dari satu nilai, misalnya daftar nama mahasiswa atau daftar barang belanja. Jika kita menggunakan array biasa, kita harus menentukan ukurannya di awal, dan ukuran itu tidak bisa berubah. Ini menjadi masalah jika kita tidak tahu berapa banyak data yang akan kita simpan saat program berjalan.

Di sinilah `ArrayList` hadir sebagai solusi. Sebagai bagian dari Java Collections Framework, `ArrayList` adalah implementasi dari antarmuka `List` yang menyediakan array yang dapat diubah ukurannya. Ini berarti Anda bisa menambah atau menghapus elemen kapan saja, dan `ArrayList` akan secara otomatis mengatur kapasitas penyimpanannya. Ia bisa menyimpan objek dari tipe data apa pun (misalnya `String`, `Integer`, objek kustom buatan Anda), namun perlu diingat bahwa ia tidak bisa secara langsung menyimpan tipe data primitif seperti `int` atau `double`; untuk itu, Java akan otomatis melakukan 'autoboxing' ke versi objeknya (`Integer`, `Double`).

Secara internal, `ArrayList` sebenarnya menggunakan array biasa sebagai wadah penyimpanannya. Namun, semua kompleksitas pengelolaan ukuran array ini disembunyikan dari programmer. Saat `ArrayList` penuh dan Anda mencoba menambahkan elemen baru, ia akan secara otomatis membuat array internal yang lebih besar (biasanya 1.5 kali ukuran sebelumnya), lalu menyalin semua elemen lama ke array baru tersebut. Ini membuat penggunaan `ArrayList` sangat fleksibel dan mudah. Metode-metode pentingnya antara lain `add()` untuk menambah, `get(index)` untuk mengakses, `set(index, value)` untuk mengubah, `remove(index)` atau `remove(object)` untuk menghapus, dan `size()` untuk mengetahui jumlah elemen saat ini.

Karena kemudahan dan fleksibilitasnya, `ArrayList` adalah salah satu kelas koleksi yang paling sering digunakan dalam aplikasi Java. Ini sangat cocok untuk skenario di mana Anda membutuhkan daftar data yang urut, dapat diakses berdasarkan indeks, dan ukurannya sering berubah. Memahami dan menguasai `ArrayList` akan sangat membantu Anda dalam mengembangkan berbagai fitur aplikasi yang memerlukan manajemen kumpulan data.

```
import java.util.ArrayList;

public class ContohArrayList {
    public static void main(String[] args) {
        // Membuat ArrayList untuk menyimpan nama (tipe String)
        ArrayList<String> daftarNama = new ArrayList<>();

        // Menambah elemen
        daftarNama.add("Budi");
        daftarNama.add("Siti");
        daftarNama.add("Andi");

        // Mengakses elemen
        System.out.println("Nama pertama: " + daftarNama.get(0)); // Output: Budi

        // Menghapus elemen
        daftarNama.remove(1); // Menghapus "Siti"

        // Mendapatkan jumlah elemen
        System.out.println("Jumlah nama sekarang: " + daftarNama.size()); // Output: 2
    }
}
```

💡 **Fun Fact:** Meskipun dinamakan 'ArrayList', secara teknis ia bukan array itu sendiri, melainkan sebuah kelas yang mengimplementasikan antarmuka `List` dengan menggunakan array sebagai penyimpanan internalnya. Setiap kali `ArrayList` perlu memperluas kapasitasnya, secara default ia akan meningkatkan ukurannya menjadi sekitar 1.5 kali dari ukuran sebelumnya.
