## ☕ Java Concept: ArrayList: Koleksi Dinamis yang Praktis

**Category:** Collections
**TL;DR:** ArrayList adalah implementasi daftar (list) berbasis array yang dinamis, memungkinkan Anda menyimpan dan mengelola urutan objek dengan mudah tanpa membatasi ukuran di awal.

Kita seringkali dihadapkan pada kebutuhan untuk menyimpan sejumlah objek yang serupa, seperti daftar nama siswa, barang belanjaan, atau catatan transaksi. Jika kita menggunakan array biasa, kita harus menentukan ukuran array di awal dan ukurannya tidak bisa diubah setelah dibuat. Ini bisa menjadi masalah jika kita tidak tahu pasti berapa banyak data yang akan kita simpan, sehingga seringkali kita harus menebak ukuran atau membuat array baru.

Di sinilah `ArrayList` hadir sebagai solusi yang elegan. `ArrayList` adalah kelas yang disediakan dalam package `java.util` di Java, yang memungkinkan kita untuk membuat daftar objek yang ukurannya dapat berubah secara dinamis. Anda tidak perlu menentukan kapasitas awal; `ArrayList` akan otomatis menyesuaikan ukurannya saat Anda menambah atau mengurangi elemen. Ini sangat memudahkan manajemen koleksi data yang jumlahnya tidak tetap.

Untuk menggunakan `ArrayList`, Anda pertama-tama perlu mengimpornya. Kemudian, Anda bisa membuat objek `ArrayList` dan menentukan tipe data apa yang akan disimpannya menggunakan *generics* (misalnya `<String>` untuk teks, atau `<Integer>` untuk angka). Menambah elemen sangat mudah dengan method `add()`. Anda juga bisa mengambil elemen pada indeks tertentu (dimulai dari 0) menggunakan method `get()` dan menghapus elemen dengan `remove()`, baik berdasarkan indeks maupun berdasarkan objeknya. Method `size()` akan memberi tahu berapa banyak elemen yang ada saat ini.

Kepraktisan `ArrayList` menjadikannya salah satu kelas koleksi yang paling sering digunakan. Ia sangat ideal untuk skenario di mana Anda sering menambah, menghapus, atau mengakses elemen, dan jumlah total elemen bisa bervariasi selama program berjalan. Kemampuannya untuk secara otomatis mengatur kapasitas membuatnya jauh lebih fleksibel dan nyaman dibandingkan array biasa untuk banyak aplikasi sehari-hari.

```
import java.util.ArrayList;

public class ContohArrayList {
    public static void main(String[] args) {
        ArrayList<String> daftarNama = new ArrayList<>(); // Membuat ArrayList untuk String
        daftarNama.add("Alice"); // Menambah elemen
        daftarNama.add("Bob");
        daftarNama.add("Charlie");

        System.out.println("Jumlah elemen: " + daftarNama.size()); // Output: 3
        System.out.println("Elemen ke-0: " + daftarNama.get(0)); // Output: Alice

        daftarNama.remove(1); // Menghapus elemen pada indeks 1 (Bob)
        System.out.println("Daftar setelah dihapus: " + daftarNama); // Output: [Alice, Charlie]
    }
}
```

💡 **Fun Fact:** Meskipun `ArrayList` bersifat dinamis, secara internal ia masih menggunakan array biasa. Ketika `ArrayList` mencapai kapasitas maksimumnya, ia akan membuat array baru yang lebih besar dan menyalin semua elemen dari array lama ke array yang baru itu. Proses ini dirancang agar efisien, tetapi memahami cara kerjanya dapat membantu Anda mengoptimalkan performa dalam kasus-kasar tertentu.
