## ☕ Java Concept: Memahami Struktur Data ArrayList

**Category:** Collections
**TL;DR:** ArrayList adalah struktur data dinamis yang memungkinkan kita menyimpan dan mengelola daftar objek dengan mudah, mirip array tetapi ukurannya bisa berubah secara otomatis.

ArrayList adalah salah satu kelas yang paling sering digunakan dalam kerangka kerja Java Collections. Bayangkan Anda perlu menyimpan daftar nama siswa, nilai ujian, atau daftar belanja. Jika menggunakan array biasa, Anda harus menentukan ukuran array sejak awal, dan akan kesulitan jika jumlah data berubah di kemudian hari. ArrayList hadir untuk mengatasi keterbatasan ini, karena ia dapat tumbuh atau menyusut secara otomatis sesuai kebutuhan.

Secara internal, ArrayList sebenarnya menggunakan array biasa untuk menyimpan elemen-elemennya. Namun, semua manajemen ukuran dan operasi penambahan/penghapusan elemen yang kompleks dilakukan di belakang layar oleh kelas ArrayList itu sendiri. Ini membuat developer bisa fokus pada logika aplikasi tanpa perlu pusing memikirkan alokasi memori atau perubahan ukuran array secara manual. Ia juga menyediakan banyak metode praktis untuk manipulasi daftar.

Untuk menggunakan ArrayList, kita perlu mendeklarasikannya dengan tipe data objek yang akan disimpan di dalamnya (menggunakan Generics, misalnya `<String>` untuk menyimpan teks, atau `<Integer>` untuk angka). Setelah itu, kita bisa menambahkan elemen menggunakan metode `add()`, mengaksesnya dengan `get(index)`, mengubahnya dengan `set(index, value)`, dan menghapusnya dengan `remove(index)` atau `remove(object)`. Fleksibilitas ini menjadikannya pilihan ideal untuk berbagai skenario di mana jumlah item dalam daftar tidak tetap.

```
import java.util.ArrayList;
import java.util.List; // Praktik terbaik menggunakan interface List

public class DemoArrayList {
    public static void main(String[] args) {
        // Membuat ArrayList untuk menyimpan nama-nama mata kuliah
        List<String> daftarMataKuliah = new ArrayList<>();

        // Menambahkan elemen
        daftarMataKuliah.add("Pemrograman Java");
        daftarMataKuliah.add("Basis Data");
        daftarMataKuliah.add("Struktur Data");

        // Mengakses elemen pertama
        System.out.println("Mata kuliah pertama: " + daftarMataKuliah.get(0));

        // Iterasi melalui semua elemen
        System.out.println("Daftar Mata Kuliah:");
        for (String mk : daftarMataKuliah) {
            System.out.println("- " + mk);
        }

        // Menghapus elemen
        daftarMataKuliah.remove("Basis Data");
        System.out.println("Setelah menghapus: " + daftarMataKuliah);
    }
}
```

💡 **Fun Fact:** Secara default, ArrayList di Java dimulai dengan kapasitas 10 elemen. Ketika kapasitas ini penuh dan Anda mencoba menambahkan elemen baru, ArrayList akan secara otomatis menambahkan ukurannya dengan membuat array baru yang lebih besar dan menyalin semua elemen lama ke array baru tersebut.
