## ☕ Java Concept: Kelas dan Objek: Fondasi Java OOP

**Category:** OOP
**TL;DR:** Kelas adalah blueprint, objek adalah instance nyata dari blueprint tersebut, membentuk dasar pemrograman berorientasi objek di Java.

Dalam pemrograman Java, konsep Kelas (Class) dan Objek (Object) adalah fondasi utama dari paradigma Object-Oriented Programming (OOP). Kelas dapat diibaratkan sebagai cetak biru (blueprint) atau template yang mendefinisikan struktur (atribut atau variabel) dan perilaku (metode atau fungsi) yang akan dimiliki oleh entitas tertentu. Misalnya, jika Anda ingin memodelkan 'Mobil', kelas 'Mobil' akan mendefinisikan atribut seperti merek, warna, tahun pembuatan, dan perilaku seperti menyalakan mesin atau mengerem.

Objek, di sisi lain, adalah instance nyata atau perwujudan konkret dari sebuah kelas. Jika kelas 'Mobil' adalah cetak birunya, maka 'Mobil saya merek Toyota warna merah tahun 2020' adalah sebuah objek dari kelas 'Mobil'. Setiap objek memiliki nilai unik untuk atribut-atributnya dan dapat memanggil metode yang didefinisikan dalam kelasnya. Anda dapat membuat banyak objek dari satu kelas yang sama, di mana setiap objek akan memiliki keadaan (state) dan identitasnya sendiri.

Hubungan antara kelas dan objek sangat penting: kelas adalah definisi abstrak, sementara objek adalah manifestasi konkret dari definisi tersebut di memori program. Untuk dapat berinteraksi dengan fungsionalitas yang didefinisikan dalam sebuah kelas (seperti atribut dan metode), kita harus terlebih dahulu membuat objek dari kelas tersebut. Proses pembuatan objek ini sering disebut sebagai instansiasi.

Pemahaman mendalam tentang Kelas dan Objek sangat krusial karena ini memungkinkan kita untuk mengatur kode menjadi unit-unit yang lebih kecil dan terkelola dengan baik (modul), mempromosikan penggunaan kembali kode (reusability), dan membuat program yang lebih mudah dipahami dan dikembangkan. Ini adalah langkah pertama untuk membangun aplikasi Java yang kompleks dan efisien.

```
class Mobil {
    String merek;
    int tahun;

    public Mobil(String merek, int tahun) {
        this.merek = merek;
        this.tahun = tahun;
    }

    public void tampilkanInfo() {
        System.out.println("Mobil: " + merek + ", Tahun: " + tahun);
    }
}

public class DemoMobil {
    public static void main(String[] args) {
        Mobil mobilSaya = new Mobil("Toyota", 2020); // Membuat objek mobilSaya
        Mobil mobilTeman = new Mobil("Honda", 2022); // Membuat objek mobilTeman

        mobilSaya.tampilkanInfo();
        mobilTeman.tampilkanInfo();
    }
}
```

💡 **Fun Fact:** Java dirancang sepenuhnya berorientasi objek dari awal, dan konsep kelas serta objek adalah inti dari setiap program Java yang berjalan. Bahkan method `main()` pun berada di dalam sebuah kelas!
