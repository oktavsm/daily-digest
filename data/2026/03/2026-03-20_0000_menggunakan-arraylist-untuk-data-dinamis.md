## ☕ Java Concept: Menggunakan ArrayList untuk Data Dinamis

**Category:** Collections
**TL;DR:** ArrayList adalah struktur data dinamis di Java yang memungkinkan kita menyimpan dan mengelola kumpulan objek dengan ukuran yang bisa berubah sesuai kebutuhan.

Di Java, array standar memiliki ukuran tetap setelah dibuat, yang bisa menjadi batasan ketika kita tidak tahu berapa banyak data yang akan disimpan atau jumlahnya sering berubah. Di sinilah `ArrayList` hadir sebagai solusi yang sangat praktis. `ArrayList` adalah bagian penting dari Java Collections Framework dan merupakan implementasi konkret dari interface `List`. Keunggulan utamanya adalah kemampuannya untuk secara otomatis menyesuaikan ukurannya saat elemen ditambahkan atau dihapus, membuatnya sangat fleksibel untuk mengelola kumpulan objek yang dinamis tanpa perlu repot dengan manajemen ukuran array secara manual.

Untuk menggunakan `ArrayList`, kita perlu mengimpornya dari paket `java.util`. Deklarasi dan inisialisasi dilakukan dengan sintaks `ArrayList<T> namaList = new ArrayList<>();`, di mana `T` adalah tipe data objek yang ingin disimpan (misalnya `String`, `Integer`, atau objek kustom buatan Anda). `ArrayList` hanya bisa menyimpan objek, bukan tipe data primitif secara langsung – jadi untuk angka, kita gunakan `Integer`, bukan `int`.

Beberapa operasi dasar yang paling sering digunakan meliputi: `add(element)` untuk menambahkan elemen baru ke akhir list, `get(index)` untuk mengambil elemen pada posisi tertentu, `set(index, element)` untuk mengubah elemen yang ada di indeks tertentu, dan `remove(index)` atau `remove(object)` untuk menghapus elemen. Contoh praktisnya, bayangkan Anda sedang membuat aplikasi manajemen daftar belanja. Jumlah item belanja bisa bertambah atau berkurang sewaktu-waktu. Menggunakan `ArrayList<String>` untuk menyimpan nama-nama item akan jauh lebih efisien daripada array biasa, karena Anda tidak perlu khawatir tentang membuang dan membuat ulang array baru setiap kali ada perubahan jumlah item. `ArrayList` juga sangat mudah diulang (iterate) menggunakan loop `for-each` yang sintaksnya ringkas.

```
import java.util.ArrayList;

public class DaftarMahasiswa {
    public static void main(String[] args) {
        ArrayList<String> namaMahasiswa = new ArrayList<>();
        namaMahasiswa.add("Budi");
        namaMahasiswa.add("Ani");
        namaMahasiswa.add("Candra");
        System.out.println("Daftar awal: " + namaMahasiswa);
        namaMahasiswa.remove("Ani"); // Menghapus Ani
        namaMahasiswa.add(0, "Deni"); // Menambah Deni di awal
        System.out.println("Daftar setelah update: " + namaMahasiswa);
        System.out.println("Mahasiswa pertama: " + namaMahasiswa.get(0));
    }
}
```

💡 **Fun Fact:** Meskipun `ArrayList` terlihat sangat fleksibel, di balik layarnya ia sebenarnya diimplementasikan menggunakan array biasa (static array). Ketika `ArrayList` mencapai kapasitas maksimumnya, ia akan membuat array baru yang lebih besar (biasanya 1.5 kali ukuran array lama) dan menyalin semua elemen dari array lama ke array baru. Proses ini disebut 'resizing'.
