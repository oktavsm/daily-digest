## 🌐 Networking Concept: Redundansi Gateway dengan HSRP

**Category:** Routing
**TL;DR:** HSRP memungkinkan dua atau lebih router berbagi satu alamat IP virtual sebagai default gateway, memastikan ketersediaan tinggi dan failover otomatis.

Untuk melengkapi konfigurasi dan memastikan router dengan prioritas lebih tinggi akan selalu menjadi Active, kita juga dapat mengaktifkan fitur *preemption*. Fitur ini memungkinkan router dengan prioritas tertinggi untuk merebut kembali peran Active setelah pulih dari kegagalan, memastikan jalur optimal selalu digunakan dan mengoptimalkan penggunaan sumber daya.

```
standby 1 ip 192.168.1.254
```

💡 **Fun Fact:** HSRP adalah protokol First Hop Redundancy Protocol (FHRP) yang dikembangkan secara proprietary oleh Cisco Systems. Ada juga versi standar terbuka yang serupa, yaitu Virtual Router Redundancy Protocol (VRRP).
