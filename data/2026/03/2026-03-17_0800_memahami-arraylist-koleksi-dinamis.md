## ☕ Java Concept: Memahami ArrayList: Koleksi Dinamis

**Category:** Collections
**TL;DR:** ArrayList adalah implementasi List yang paling umum, memungkinkan penyimpanan koleksi objek yang ukurannya bisa berubah secara dinamis.

Saat memprogram, seringkali kita perlu menyimpan sekumpulan data, namun kita tidak tahu pasti berapa banyak data yang akan ada di awal. Array biasa di Java memiliki ukuran tetap yang ditentukan saat dibuat, sehingga merepotkan jika kita perlu menambah atau mengurangi elemen setelah array dibuat. Di sinilah `ArrayList` hadir sebagai solusi. `ArrayList` adalah bagian dari Java Collections Framework yang menyediakan array yang "dinamis", artinya ukurannya dapat bertambah atau berkurang sesuai kebutuhan.

Menggunakan `ArrayList` sangat mudah. Anda bisa membuatnya dengan `new ArrayList<TipeData>()`. Untuk menambahkan elemen, gunakan metode `add()`. Untuk mengambil elemen pada posisi tertentu (indeks dimulai dari 0), gunakan `get(index)`. Dan jika Anda ingin menghapus elemen, ada `remove(index)` atau `remove(objek)`. Ini semua membuat manipulasi koleksi data menjadi jauh lebih fleksibel dibandingkan array statis, memungkinkan Anda untuk menambahkan atau menghapus elemen kapan saja tanpa perlu khawatir tentang kapasitas awal.

Salah satu fitur penting `ArrayList` adalah penggunaan Generics, yang ditunjukkan dengan `<TipeData>` saat Anda mendeklarasikannya, misalnya `ArrayList<String>`. Generics memastikan bahwa `ArrayList` Anda hanya akan menyimpan objek dari tipe data yang spesifik, atau subtipe dari tipe tersebut. Ini sangat meningkatkan keamanan tipe (type safety) program Anda karena compiler dapat mendeteksi kesalahan tipe pada saat kompilasi, bukan saat runtime, dan juga menghilangkan kebutuhan untuk melakukan casting manual setiap kali Anda mengambil elemen. Ingat, `ArrayList` menyimpan objek, jadi untuk tipe primitif seperti `int` atau `double`, Anda perlu menggunakan versi wrapper-nya (misalnya `Integer` atau `Double`).

`ArrayList` adalah salah satu struktur data yang paling sering digunakan dalam pengembangan aplikasi Java sehari-hari. Kemudahannya dalam menangani koleksi data yang ukurannya tidak pasti, dikombinasikan dengan keamanan tipe dari Generics, menjadikannya pilihan utama untuk berbagai skenario, mulai dari menyimpan daftar nama pengguna, daftar item belanja, hingga log transaksi. Memahami `ArrayList` adalah langkah krusial dalam menguasai penanganan koleksi data di Java.

```
import java.util.ArrayList;

// Contoh penggunaan dalam sebuah metode atau main:
ArrayList<String> daftarBuah = new ArrayList<>();
daftarBuah.add("Apel");
daftarBuah.add("Jeruk");
daftarBuah.add("Mangga");

System.out.println("Buah pertama: " + daftarBuah.get(0)); // Output: Apel

daftarBuah.remove("Jeruk"); // Menghapus "Jeruk"
System.out.println("Jumlah buah: " + daftarBuah.size()); // Output: 2

for (String buah : daftarBuah) { // Iterasi
    System.out.println("Ada buah: " + buah);
}
```

💡 **Fun Fact:** Faktanya, `ArrayList` diimplementasikan menggunakan array biasa di balik layar. Namun, ia secara otomatis mengatur ukurannya dengan membuat array baru yang lebih besar dan menyalin elemen lama saat kapasitasnya penuh, sehingga memberikan kesan "dinamis".
