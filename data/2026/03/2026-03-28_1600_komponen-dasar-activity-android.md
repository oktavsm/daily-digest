## 🤖 Android Concept: Komponen Dasar: Activity Android

**Category:** UI
**TL;DR:** Activity adalah blok bangunan fundamental yang menampilkan satu layar antarmuka pengguna, memungkinkan interaksi pengguna dengan aplikasi Anda.

Dalam pengembangan aplikasi Android, Activity adalah komponen inti yang menyediakan layar tunggal dengan antarmuka pengguna (UI) yang dapat diinteraksikan. Bayangkan setiap layar yang Anda lihat di aplikasi (seperti layar login, daftar belanja, atau detail produk) sebagai sebuah Activity yang terpisah. Activity bertanggung jawab untuk mengelola visual layout dan merespons interaksi pengguna, menjadikannya 'jendela' utama aplikasi Anda ke dunia pengguna.

Setiap Activity memiliki siklus hidup (lifecycle) yang terdefinisi dengan baik, mulai dari dibuat hingga dihancurkan. Beberapa metode penting dalam siklus hidup ini antara lain `onCreate()`, `onStart()`, `onResume()`, `onPause()`, `onStop()`, dan `onDestroy()`. Metode `onCreate()` adalah yang paling penting saat Anda memulai, di mana Anda biasanya melakukan inisialisasi awal seperti mengatur tampilan UI menggunakan `setContentView()` dengan layout XML yang telah Anda desain. Memahami siklus hidup ini krusial untuk mengelola status aplikasi dan sumber daya dengan benar.

Activity tidak hanya menampilkan UI, tetapi juga berperan sebagai titik masuk untuk interaksi yang lebih kompleks. Anda dapat memulai Activity lain dari Activity yang sedang berjalan menggunakan `Intent`, sebuah objek yang menjelaskan 'niat' untuk melakukan sesuatu, seperti membuka Activity baru, mengirim email, atau menampilkan peta. Ini memungkinkan aplikasi Android untuk terdiri dari banyak Activity yang bekerja sama untuk membentuk pengalaman pengguna yang kaya dan mulus.

```
package com.example.myfirstapp

import android.os.Bundle
import androidx.appcompat.app.AppCompatActivity

class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main) // Menghubungkan layout XML
        // Di sini Anda bisa inisialisasi UI komponen lain
    }
}
```

💡 **Fun Fact:** Android menggunakan 'back stack' untuk mengelola Activity. Setiap kali Anda meluncurkan Activity baru, ia diletakkan di atas tumpukan. Ketika Anda menekan tombol 'Back', Activity teratas akan dihancurkan dan Activity di bawahnya akan muncul kembali.
