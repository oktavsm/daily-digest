## 🤖 Android Concept: Activity: Jendela Utama Aplikasi

**Category:** UI
**TL;DR:** Activity adalah komponen dasar yang menyediakan satu layar antarmuka pengguna untuk berinteraksi dengan pengguna.

Activity adalah "jendela" atau "layar" tunggal yang berinteraksi dengan pengguna. Setiap aplikasi Android terdiri dari satu atau lebih Activity. Ketika Anda membuka sebuah aplikasi, biasanya Anda memulai dengan sebuah Activity utama. Ia bertanggung jawab untuk menampilkan tata letak (layout) UI, menangani event dari pengguna, dan mengelola siklus hidupnya sendiri.

Secara teknis, Activity adalah subclass dari `android.app.Activity` (atau biasanya `AppCompatActivity` untuk kompatibilitas ke belakang). Ia bertindak sebagai wadah untuk komponen UI seperti tombol, teks, gambar, dan elemen lainnya. Activity berfungsi sebagai titik masuk utama bagi pengguna untuk berinteraksi dengan fitur-fitur spesifik aplikasi Anda.

Setiap Activity memiliki siklus hidup (lifecycle) yang terdefinisi dengan baik, mulai dari dibuat, aktif, hingga dihancurkan. Metode-metode callback seperti `onCreate()`, `onStart()`, `onResume()`, `onPause()`, `onStop()`, dan `onDestroy()` memungkinkan Anda untuk menjalankan kode pada tahapan tertentu dari siklus hidup Activity, memastikan bahwa sumber daya dikelola dengan baik dan pengalaman pengguna lancar.

Memahami Activity adalah kunci untuk membangun aplikasi Android. Ini adalah fondasi di mana komponen UI lainnya seperti Fragment dibangun, dan merupakan titik awal untuk mengorganisir alur aplikasi Anda.

```
package com.example.myfirstapp

import android.os.Bundle
import androidx.appcompat.app.AppCompatActivity

class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main) // Menghubungkan Activity dengan layout XML
    }
}
```

💡 **Fun Fact:** Activity pertama kali diperkenalkan dengan Android 1.0 pada tahun 2008. Setiap aplikasi Android setidaknya harus memiliki satu Activity untuk dapat diluncurkan.
