## 🤖 Android Concept: Activity: Fondasi Aplikasi Android

**Category:** UI
**TL;DR:** Activity adalah komponen dasar yang menyediakan layar tunggal tempat pengguna berinteraksi dengan aplikasi Android.

Setiap aplikasi Android terdiri dari satu atau lebih Activity. Anggap saja Activity sebagai sebuah "layar" dalam aplikasi Anda, misalnya layar login, layar daftar produk, atau layar pengaturan. Activity bertanggung jawab untuk menampilkan antarmuka pengguna (UI) dan merespons interaksi pengguna pada layar tersebut. Saat Anda membuka sebuah aplikasi, Activity pertamalah yang akan ditampilkan kepada Anda.

Activity memiliki siklus hidup (lifecycle) yang terdefinisi dengan baik, yang mencerminkan berbagai status yang dapat dilaluinya, seperti saat Activity dibuat (onCreate), terlihat oleh pengguna (onStart), atau berada di latar depan dan siap berinteraksi (onResume). Memahami siklus hidup ini sangat penting untuk mengelola sumber daya dan perilaku aplikasi agar tetap responsif dan efisien.

Untuk membuat Activity baru, Anda perlu mendefinisikannya dalam file Kotlin dan mendeklarasikannya di file `AndroidManifest.xml` aplikasi Anda. Navigasi antar Activity biasanya dilakukan menggunakan objek `Intent`, yang bertindak sebagai "pesan" untuk meminta sistem Android memulai komponen aplikasi lain, termasuk Activity. Ini memungkinkan aplikasi berpindah dari satu layar ke layar lainnya dengan mulus.

Secara ringkas, Activity adalah blok bangunan fundamental untuk setiap UI aplikasi Android Anda. Tanpa Activity, tidak akan ada layar untuk menampilkan konten atau berinteraksi dengan pengguna. Dengan memahami bagaimana Activity bekerja, Anda telah mengambil langkah pertama yang krusial dalam mengembangkan aplikasi Android yang fungsional dan menarik.

```
package com.example.myapp

import android.os.Bundle
import androidx.appcompat.app.AppCompatActivity

class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main) // Menghubungkan layout XML
    }
}
```

💡 **Fun Fact:** Activity pertama kali diperkenalkan bersamaan dengan rilis Android 1.0 pada tahun 2008. Nama 'Activity' dipilih untuk menggambarkan bahwa ini adalah tempat di mana pengguna melakukan 'aktivitas' tertentu dalam aplikasi.
