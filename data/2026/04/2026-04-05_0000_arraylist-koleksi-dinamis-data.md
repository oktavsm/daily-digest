## ☕ Java Concept: ArrayList: Koleksi Dinamis Data

**Category:** Collections
**TL;DR:** ArrayList adalah struktur data di Java yang memungkinkan kita menyimpan daftar objek dengan ukuran yang bisa berubah secara dinamis.

Ketika kita membuat array standar di Java, ukurannya sudah ditentukan di awal dan tidak bisa diubah setelahnya. Ini seringkali menjadi kendala jika kita tidak tahu pasti berapa banyak data yang akan kita simpan. Di sinilah `ArrayList` hadir sebagai solusi. `ArrayList` adalah bagian dari Java Collections Framework yang menyediakan implementasi daftar (list) yang ukurannya bisa bertambah atau berkurang secara dinamis sesuai kebutuhan, membuatnya sangat fleksibel untuk mengelola kumpulan objek.

Untuk mulai menggunakan `ArrayList`, kita perlu mengimpornya dari paket `java.util`. Deklarasinya mirip dengan array, tetapi menggunakan kurung sudut (`<>`) untuk menentukan tipe data objek yang akan disimpan (disebut Generics, akan dipelajari lebih lanjut). Setelah dibuat, kita bisa menambahkan elemen menggunakan method `add()`. Misalnya, `ArrayList<String> daftarNama = new ArrayList<>();` akan membuat daftar nama, dan `daftarNama.add("Budi");` akan menambahkan nama 'Budi' ke dalamnya.

Mengakses elemen dalam `ArrayList` juga mudah. Kita bisa mendapatkan elemen pada indeks tertentu menggunakan method `get()`, mirip dengan array. Ingat, indeks dimulai dari 0. Untuk mengetahui berapa banyak elemen yang ada di dalam `ArrayList`, kita bisa menggunakan method `size()`. Jika kita ingin menghapus elemen, kita bisa menggunakan method `remove()` dengan memberikan indeks atau langsung objek yang ingin dihapus. Fleksibilitas ini adalah alasan utama mengapa `ArrayList` sering digunakan.

Secara praktis, `ArrayList` sangat sering digunakan untuk berbagai skenario, seperti menyimpan daftar item belanja, daftar siswa dalam sebuah kelas, atau kumpulan data dari database sebelum diproses. Kemampuannya untuk secara otomatis menyesuaikan ukuran sangat menyederhanakan kode dan mengurangi kebutuhan untuk penanganan ukuran array secara manual, membuat program lebih efisien dan mudah dikelola.

```
import java.util.ArrayList;

public class ContohArrayList {
    public static void main(String[] args) {
        ArrayList<String> daftarNama = new ArrayList<>();
        daftarNama.add("Alice");
        daftarNama.add("Bob");
        daftarNama.add("Charlie");

        System.out.println("Elemen ke-0: " + daftarNama.get(0));
        System.out.println("Ukuran daftar: " + daftarNama.size());
        daftarNama.remove(1); // Menghapus "Bob"
        System.out.println("Daftar sekarang: " + daftarNama);
    }
}
```

💡 **Fun Fact:** Tahukah Anda bahwa di balik layar, `ArrayList` sebenarnya menggunakan array biasa? Setiap kali Anda menambahkan elemen dan array internalnya penuh, `ArrayList` akan membuat array baru yang lebih besar (biasanya 1.5 kali ukuran semula) dan menyalin semua elemen lama ke array baru tersebut secara otomatis. Inilah rahasia di balik 'ukuran dinamis'nya!
