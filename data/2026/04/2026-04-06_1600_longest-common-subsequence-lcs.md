## ⚡ Competitive Programming Concept: Longest Common Subsequence (LCS)

**Category:** DP
**TL;DR:** Algoritma Longest Common Subsequence (LCS) mencari urutan karakter terpanjang yang muncul di dua string, tidak harus berurutan secara contiguous.

LCS adalah masalah klasik dalam pemrograman dinamis yang bertujuan menemukan subsequence terpanjang yang sama dari dua string input. Subsequence berarti kita bisa menghapus nol atau lebih karakter dari string asli, tetapi urutan karakter yang tersisa harus dipertahankan. Sebagai contoh, "ACE" adalah subsequence dari "ABCDE", tapi "AEC" bukan.

Untuk menyelesaikan LCS, kita menggunakan pendekatan pemrograman dinamis. Kita mendefinisikan `dp[i][j]` sebagai panjang LCS dari prefiks pertama string `S1` yang berukuran `i` (yaitu `S1[0...i-1]`) dan prefiks pertama string `S2` yang berukuran `j` (yaitu `S2[0...j-1]`). Tabel DP ini akan kita isi secara iteratif.

Rekurensi untuk `dp[i][j]` bergantung pada karakter `S1[i-1]` dan `S2[j-1]`. Jika `S1[i-1] == S2[j-1]` (karakter terakhir prefiks cocok), maka karakter ini dapat menjadi bagian dari LCS, sehingga `dp[i][j] = 1 + dp[i-1][j-1]`. Jika `S1[i-1] != S2[j-1]` (karakter terakhir prefiks tidak cocok), maka kita harus memilih opsi terbaik dari dua kemungkinan: menghilangkan `S1[i-1]` atau menghilangkan `S2[j-1]`. Jadi, `dp[i][j] = max(dp[i-1][j], dp[i][j-1])`. Kasus dasar adalah `dp[0][j] = 0` dan `dp[i][0] = 0` karena LCS dengan string kosong selalu 0.

Setelah tabel DP terisi, `dp[N][M]` (dengan N dan M adalah panjang `S1` dan `S2` masing-masing) akan berisi panjang LCS total. Kompleksitas waktu untuk membangun tabel DP berukuran `N x M` adalah O(N*M) karena setiap sel diisi dalam waktu O(1). Kompleksitas ruang juga O(N*M) untuk menyimpan tabel DP, meskipun bisa dioptimalkan menjadi O(min(N, M)) jika kita hanya membutuhkan dua baris/kolom terakhir.

```
int lcs(const std::string& s1, const std::string& s2) {
    int n = s1.length();
    int m = s2.length();
    std::vector<std::vector<int>> dp(n + 1, std::vector<int>(m + 1, 0));

    for (int i = 1; i <= n; ++i) {
        for (int j = 1; j <= m; ++j) {
            if (s1[i - 1] == s2[j - 1]) {
                dp[i][j] = 1 + dp[i - 1][j - 1];
            } else {
                dp[i][j] = std::max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
    }
    return dp[n][m];
}
```
