## ☕ Java Concept: Kelas dan Objek: Dasar OOP

**Category:** OOP
**TL;DR:** Java dibangun di atas konsep Kelas sebagai cetak biru dan Objek sebagai hasil cetakan, membentuk dasar pemrograman berorientasi objek.

Dalam pemrograman Java, pondasi utama adalah konsep Object-Oriented Programming (OOP), dan dua pilar utamanya adalah Kelas (Class) serta Objek (Object). Bayangkan sebuah Kelas sebagai sebuah cetak biru atau *template* untuk menciptakan sesuatu. Misalnya, jika kita ingin membuat banyak mobil dengan karakteristik yang sama, kita tidak perlu mendefinisikan setiap mobil satu per satu dari nol. Cukup buat satu "cetak biru Mobil" yang mendefinisikan apa saja yang dimiliki mobil (warna, merek, kecepatan) dan apa saja yang bisa dilakukan mobil (melaju, mengerem).

Sebuah Objek adalah instansi nyata atau hasil jadi dari cetak biru Kelas tersebut. Jadi, dari cetak biru "Mobil" tadi, kita bisa menciptakan Objek "mobilSaya" berwarna merah, merek Toyota, dan "mobilTeman" berwarna biru, merek Honda. Setiap Objek memiliki atribut (data atau karakteristik, seperti warna dan merek) yang ditentukan oleh Kelasnya, dan dapat melakukan aksi atau perilaku (melaju, mengerem) melalui metode (method) yang juga didefinisikan dalam Kelasnya. Setiap objek adalah entitas independen yang memiliki keadaannya sendiri.

Manfaat utama dari penggunaan Kelas dan Objek adalah untuk mengorganisir kode program agar lebih modular, mudah dipahami, dan dapat digunakan kembali (reusable). Daripada menulis kode yang panjang dan tidak terstruktur, kita bisa memecah masalah besar menjadi bagian-bagian kecil yang direpresentasikan sebagai objek. Ini memudahkan kolaborasi dalam tim, pemeliharaan kode di masa depan, serta skalabilitas aplikasi saat fitur-fitur baru ditambahkan.

```
class Mobil {
    String merek;
    String warna;

    void nyalakanMesin() {
        System.out.println(merek + " berwarna " + warna + " menyala.");
    }
}

public class Main {
    public static void main(String[] args) {
        Mobil mobilSaya = new Mobil(); // Membuat objek dari kelas Mobil
        mobilSaya.merek = "Toyota";
        mobilSaya.warna = "Merah";
        mobilSaya.nyalakanMesin(); // Memanggil method objek
    }
}
```

💡 **Fun Fact:** Konsep OOP seperti Kelas dan Objek mulai populer pada tahun 1960-an dengan bahasa pemrograman Simula, jauh sebelum Java diciptakan. Filosofinya adalah memodelkan dunia nyata ke dalam kode, membuat program lebih intuitif dan terstruktur.
