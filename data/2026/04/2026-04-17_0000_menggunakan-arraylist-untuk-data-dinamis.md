## ☕ Java Concept: Menggunakan ArrayList untuk Data Dinamis

**Category:** Collections
**TL;DR:** ArrayList adalah struktur data dinamis yang fleksibel untuk menyimpan daftar objek, mirip array tetapi ukurannya bisa berubah secara otomatis.

ArrayList adalah salah satu implementasi paling populer dari antarmuka `List` dalam Java Collections Framework. Bayangkan Anda memiliki daftar nama mahasiswa atau daftar item belanja yang ukurannya bisa bertambah atau berkurang seiring waktu. Array tradisional (`String[] nama`) memiliki ukuran yang tetap setelah dibuat, yang bisa merepotkan jika jumlah data tidak pasti. ArrayList mengatasi batasan ini dengan memungkinkan Anda menyimpan objek dan secara otomatis menyesuaikan kapasitasnya sesuai kebutuhan.

Untuk menggunakan ArrayList, Anda perlu mengimpor `java.util.ArrayList`. Saat membuatnya, Anda biasanya menentukan tipe objek yang akan disimpan di dalamnya menggunakan *generics*, misalnya `ArrayList<String>` untuk menyimpan string, atau `ArrayList<Integer>` untuk bilangan bulat. Penentuan tipe ini membantu Java memastikan keamanan tipe (type safety) saat kompilasi. Setelah itu, Anda bisa menambahkan elemen menggunakan metode `add()`, di mana elemen akan ditambahkan di akhir daftar secara berurutan.

Setiap elemen dalam ArrayList memiliki indeks berbasis nol, sama seperti array. Anda bisa mengambil elemen pada indeks tertentu dengan metode `get()`. Jika Anda ingin mengubah elemen yang sudah ada, gunakan `set()` dengan menentukan indeks dan nilai baru. Untuk menghapus elemen, Anda bisa menggunakan `remove()` dengan menentukan indeks atau objeknya. Metode `size()` juga berguna untuk mengetahui jumlah elemen yang sedang ada di dalam ArrayList. Fleksibilitas ini menjadikan ArrayList pilihan yang sangat praktis untuk banyak skenario di mana Anda membutuhkan koleksi data yang ukurannya tidak tetap.

```
import java.util.ArrayList;

public class DaftarMahasiswa {
    public static void main(String[] args) {
        ArrayList<String> namaMahasiswa = new ArrayList<>();
        namaMahasiswa.add("Budi");
        namaMahasiswa.add("Ani");
        namaMahasiswa.add("Cici");
        System.out.println("Mahasiswa pertama: " + namaMahasiswa.get(0));
        namaMahasiswa.remove(1); // Menghapus "Ani"
        System.out.println("Jumlah mahasiswa sekarang: " + namaMahasiswa.size());
    }
}
```

💡 **Fun Fact:** Java Collections Framework, termasuk ArrayList, dirancang oleh Joshua Bloch, salah satu arsitek dan pengembang Java paling berpengaruh, yang juga menulis buku 'Effective Java'.
