## ☕ Java Concept: Mengelola Daftar Data Dinamis dengan ArrayList

**Category:** Collections
**TL;DR:** ArrayList adalah struktur data dinamis di Java yang memungkinkan Anda menyimpan dan mengelola koleksi objek dengan ukuran yang dapat berubah selama program berjalan.

Dalam pemrograman, kita seringkali perlu menyimpan banyak data yang sejenis. Array biasa (misalnya `String[] names = new String[5];`) memungkinkan kita melakukan ini, namun memiliki batasan penting: ukurannya tetap setelah dibuat. Jika Anda awalnya perlu menyimpan 5 nama tapi kemudian ternyata ada 10, Anda harus membuat array baru dan menyalin semua data. Di sinilah `ArrayList` datang sebagai solusi. `ArrayList` adalah salah satu implementasi struktur data 'list' di Java yang memungkinkan kita menyimpan daftar objek dengan ukuran yang fleksibel atau dinamis.

`ArrayList` merupakan bagian dari Java Collections Framework. Untuk menggunakannya, Anda perlu mengimpor `java.util.ArrayList`. Saat membuat `ArrayList`, Anda harus menentukan tipe data objek yang akan disimpan di dalamnya menggunakan *generics*, misalnya `<String>` untuk menyimpan teks, atau `<Integer>` untuk angka. Contohnya: `ArrayList<String> daftarNama = new ArrayList<>();`. Setelah dibuat, Anda bisa menambahkan elemen dengan metode `add()`, seperti `daftarNama.add("Alice");`. Elemen akan ditambahkan di akhir daftar secara berurutan.

Selain `add()`, `ArrayList` menyediakan banyak metode berguna lainnya. Anda bisa mengambil elemen pada posisi tertentu menggunakan `get(index)`, misalnya `daftarNama.get(0)` akan mengembalikan elemen pertama (ingat, indeks dimulai dari 0). Untuk mengubah elemen, gunakan `set(index, newElement)`. Jika Anda ingin menghapus elemen, gunakan `remove(index)` atau `remove(object)`. Untuk mengetahui berapa banyak elemen yang ada di dalam `ArrayList`, panggil metode `size()`. Ini sangat membantu ketika Anda perlu mengulang (loop) melalui semua elemen di dalamnya.

`ArrayList` sangat praktis ketika jumlah data yang akan Anda proses tidak pasti atau sering berubah selama program berjalan. Ini adalah pilihan yang sangat populer untuk banyak aplikasi, dari menyimpan daftar belanja hingga mengelola daftar pengguna. Meskipun sedikit lebih lambat dari array biasa dalam beberapa skenario karena fleksibilitasnya, kemudahan penggunaan dan kemampuannya untuk beradaptasi dengan perubahan ukuran data menjadikannya pilihan utama bagi sebagian besar pengembang Java.

```
import java.util.ArrayList;

public class ContohArrayList {
    public static void main(String[] args) {
        ArrayList<String> daftarNama = new ArrayList<>();
        daftarNama.add("Alice");
        daftarNama.add("Bob");
        daftarNama.add("Charlie");

        System.out.println("Nama pertama: " + daftarNama.get(0));
        daftarNama.remove(1); // Menghapus Bob (pada indeks 1)

        System.out.println("Jumlah nama: " + daftarNama.size());
        for (String nama : daftarNama) {
            System.out.println(nama);
        }
    }
}
```

💡 **Fun Fact:** Tahukah Anda bahwa `ArrayList` di Java sebenarnya menggunakan array biasa di "belakang layar"? Ketika `ArrayList` penuh dan Anda mencoba menambahkan elemen baru, ia akan secara otomatis membuat array internal baru yang ukurannya lebih besar, lalu menyalin semua elemen lama ke array baru tersebut. Ini yang membuatnya dinamis!
