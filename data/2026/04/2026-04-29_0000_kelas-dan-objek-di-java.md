## ☕ Java Concept: Kelas dan Objek di Java

**Category:** OOP
**TL;DR:** Kelas adalah cetak biru untuk membuat objek, sedangkan objek adalah instansi nyata dari kelas yang memiliki data (state) dan perilaku (behavior) sendiri.

Mahasiswa sekalian, mari kita mulai hari ini dengan fondasi utama pemrograman Java: Kelas (Class). Bayangkan sebuah kelas seperti cetak biru atau resep kue. Cetak biru rumah tidak bisa dihuni, dan resep kue tidak bisa dimakan, bukan? Keduanya hanyalah deskripsi atau pola. Dalam Java, sebuah `class` adalah cetak biru yang mendefinisikan struktur untuk objek: apa saja data (sering disebut *atribut* atau *field*) yang akan dimilikinya dan aksi apa saja (sering disebut *metode* atau *fungsi*) yang bisa dilakukannya. Kelas mendefinisikan bagaimana sebuah tipe data kustom kita akan terlihat dan berperilaku.

Nah, jika kelas adalah cetak biru, maka Objek (Object) adalah hasil nyatanya. Kembali ke analogi tadi, objek adalah rumah yang sudah jadi dari cetak biru tersebut, atau kue yang sudah dipanggang dari resep. Setiap objek adalah *instance* atau wujud nyata dari sebuah kelas, yang memiliki nilai-nilai spesifik untuk atribut-atribut yang didefinisikan dalam kelasnya. Misalnya, jika ada kelas `Mobil`, Anda bisa membuat banyak objek mobil: `mobilSaya` dengan merek "Honda" dan `mobilTeman` dengan merek "Toyota". Masing-masing objek `Mobil` ini adalah entitas independen yang hidup di memori program Anda.

Mengapa konsep ini begitu fundamental? Karena Java adalah bahasa pemrograman berorientasi objek (Object-Oriented Programming - OOP). Dengan kelas dan objek, kita bisa memodelkan entitas-entitas dari dunia nyata ke dalam kode program kita secara terstruktur. Ini memudahkan kita untuk mengelola kompleksitas, membuat kode lebih modular, lebih mudah dipahami, digunakan kembali (reusability), dan dipelihara. Pemahaman yang kuat tentang bagaimana mendefinisikan kelas dan membuat objek adalah kunci untuk membangun aplikasi Java yang efektif dan efisien.

```
class Mobil { // Ini adalah sebuah Kelas
    String merek;       // Ini adalah atribut (data)
    int kecepatan;      // Atribut lainnya

    void gas() {        // Ini adalah metode (perilaku)
        kecepatan += 10;
        System.out.println("Mobil " + merek + " melaju " + kecepatan + " km/jam.");
    }
}
```

💡 **Fun Fact:** Java adalah salah satu bahasa pemrograman berorientasi objek murni yang paling populer, dan konsep Kelas & Objek adalah jantung dari filosofi desainnya. Sebagian besar fitur Java lainnya, seperti *inheritance* dan *polimorfisme*, dibangun di atas dasar ini.
