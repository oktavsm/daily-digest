## ⚡ Competitive Programming Concept: Longest Common Subsequence (LCS)

**Category:** DP
**TL;DR:** Mencari urutan karakter terpanjang yang muncul di dua string dengan urutan yang sama, tidak harus berurutan (contiguous).

LCS adalah masalah klasik dalam pemrograman dinamis yang bertujuan menemukan subsequence terpanjang yang sama antara dua string. Subsequence adalah urutan karakter yang dapat diperoleh dari string asli dengan menghapus nol atau lebih karakter yang tersisa tanpa mengubah urutan karakter sisanya. Misalnya, "ACE" adalah subsequence dari "ABCDE". Tantangannya adalah menemukan subsequence terpanjang yang muncul di kedua string.

Kita dapat menyelesaikan masalah LCS menggunakan pendekatan pemrograman dinamis. Misalkan kita memiliki dua string `S1` dengan panjang `m` dan `S2` dengan panjang `n`. Kita definisikan `dp[i][j]` sebagai panjang LCS dari prefix `S1[0...i-1]` dan `S2[0...j-1]`. Sebagai base case, `dp[0][j] = 0` dan `dp[i][0] = 0` karena LCS dengan string kosong selalu 0. Untuk rekurensi:
*   Jika `S1[i-1] == S2[j-1]` (karakter terakhir sama), maka karakter ini pasti bagian dari LCS. Jadi, `dp[i][j] = 1 + dp[i-1][j-1]`.
*   Jika `S1[i-1] != S2[j-1]`, maka kita tidak bisa menyertakan kedua karakter ini. Kita harus memilih untuk "mengabaikan" salah satu karakter dan mencari LCS dari dua kemungkinan subproblem: `dp[i-1][j]` (mengabaikan `S1[i-1]`) atau `dp[i][j-1]` (mengabaikan `S2[j-1]`). Jadi, `dp[i][j] = max(dp[i-1][j], dp[i][j-1]).`

Kita membangun tabel `dp` berukuran `(m+1) x (n+1)` secara bottom-up. Iterasi melalui semua `i` dari 1 hingga `m` dan `j` dari 1 hingga `n` untuk mengisi tabel. Nilai akhir `dp[m][n]` akan menjadi panjang LCS dari seluruh `S1` dan `S2`. Kompleksitas waktu untuk mengisi tabel ini adalah `O(m * n)` karena kita melakukan operasi konstan untuk setiap sel. Kompleksitas ruang juga `O(m * n)` untuk menyimpan tabel DP, meskipun bisa dioptimalkan menjadi `O(min(m, n))` jika kita hanya tertarik pada panjang LCS dan tidak perlu membangun kembali string LCS-nya.

```
int solveLCS(const std::string& s1, const std::string& s2) {
    int m = s1.length(), n = s2.length();
    std::vector<std::vector<int>> dp(m + 1, std::vector<int>(n + 1, 0));
    for (int i = 1; i <= m; ++i) {
        for (int j = 1; j <= n; ++j) {
            if (s1[i - 1] == s2[j - 1]) dp[i][j] = 1 + dp[i - 1][j - 1];
            else dp[i][j] = std::max(dp[i - 1][j], dp[i][j - 1]);
        }
    }
    return dp[m][n];
}
```
