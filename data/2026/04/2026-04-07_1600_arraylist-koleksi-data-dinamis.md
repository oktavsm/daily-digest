## ☕ Java Concept: ArrayList: Koleksi Data Dinamis

**Category:** Collections
**TL;DR:** ArrayList adalah implementasi List yang dinamis, memungkinkan kita menyimpan dan mengelola urutan objek dengan ukuran yang bisa berubah secara fleksibel sesuai kebutuhan.

Di Java, kita sering berurusan dengan kumpulan data. Array standar memang berguna, tetapi memiliki batasan utama: ukurannya tetap setelah dibuat. Bagaimana jika kita tidak tahu berapa banyak data yang akan kita simpan, atau jika jumlah data sering berubah? Di sinilah `ArrayList` hadir sebagai solusi. `ArrayList` adalah bagian dari Java Collections Framework dan merupakan implementasi dari interface `List` yang memungkinkan kita menyimpan urutan objek dengan ukuran yang dapat berubah secara dinamis.

Menggunakan `ArrayList` sangat mudah. Untuk membuatnya, kita cukup mendeklarasikan objek `ArrayList` dan menentukan tipe data objek yang akan disimpan di dalamnya (menggunakan Generics untuk keamanan tipe). Kita bisa menambahkan elemen menggunakan metode `add()`, mengakses elemen berdasarkan indeks menggunakan `get()`, dan bahkan mengubah elemen di posisi tertentu dengan `set()`. `ArrayList` juga mempertahankan urutan penambahan elemen, artinya elemen yang Anda tambahkan pertama akan tetap di indeks 0, yang kedua di indeks 1, dan seterusnya.

Selain operasi dasar tersebut, `ArrayList` juga menyediakan berbagai metode lain yang sangat praktis. Kita bisa menghapus elemen berdasarkan indeks atau objek itu sendiri menggunakan `remove()`, mengetahui berapa banyak elemen yang ada di daftar dengan `size()`, dan memeriksa apakah daftar kosong menggunakan `isEmpty()`. Kita juga bisa dengan mudah mengulang (mengiterasi) seluruh elemen dalam `ArrayList`, misalnya menggunakan perulangan `for-each` yang sangat ringkas untuk memproses setiap elemen dalam daftar.

Penting untuk diingat bahwa `ArrayList` menyimpan objek, jadi untuk tipe data primitif seperti `int` atau `double`, Java akan secara otomatis melakukan "autoboxing" (mengubahnya menjadi objek `Integer` atau `Double`). `ArrayList` adalah pilihan yang sangat baik ketika Anda membutuhkan daftar yang fleksibel, yang elemennya sering ditambahkan atau dihapus, dan urutannya penting. Ini adalah salah satu struktur data yang paling sering digunakan dalam pengembangan aplikasi Java modern.

```
import java.util.ArrayList;

public class DemoArrayList {
    public static void main(String[] args) {
        ArrayList<String> daftarSiswa = new ArrayList<>(); // Deklarasi ArrayList
        daftarSiswa.add("Andi"); daftarSiswa.add("Budi"); // Menambah elemen
        System.out.println("Daftar: " + daftarSiswa + ", ukuran: " + daftarSiswa.size());
        System.out.println("Siswa pertama: " + daftarSiswa.get(0));
        daftarSiswa.remove(1); // Menghapus Budi (indeks 1)
        System.out.println("Daftar setelah hapus: " + daftarSiswa);
    }
}
```

💡 **Fun Fact:** Tahukah kamu bahwa ArrayList, meskipun dinamis, sebenarnya didukung oleh array biasa di dalamnya? Saat ArrayList penuh dan Anda menambahkan elemen baru, ia akan membuat array baru yang lebih besar (biasanya 1.5 kali ukuran lama) dan menyalin semua elemen lama ke array baru tersebut secara otomatis.
