## 🤖 Android Concept: Memahami Android Activity

**Category:** UI
**TL;DR:** Activity adalah komponen dasar yang merepresentasikan sebuah layar tunggal dengan antarmuka pengguna (UI) tempat pengguna berinteraksi dengan aplikasi Anda.

Dalam pengembangan Android, `Activity` adalah blok bangunan paling fundamental. Bayangkan aplikasi Anda memiliki beberapa 'layar' berbeda—misalnya, layar login, daftar produk, atau detail item. Setiap 'layar' ini pada dasarnya diimplementasikan sebagai sebuah `Activity`. Ini adalah kelas tempat Anda akan mendefinisikan UI aplikasi Anda dan semua logika yang diperlukan untuk interaksi pengguna pada layar tersebut.

Untuk membuat sebuah `Activity`, Anda biasanya akan membuat kelas Kotlin baru yang meng-extend `AppCompatActivity`, sebuah kelas dari Android Jetpack yang menyediakan kompatibilitas mundur untuk fitur-fitur UI. Setiap `Activity` memiliki siklus hidup (lifecycle) yang terdiri dari beberapa metode, seperti `onCreate()`, `onStart()`, `onResume()`, dll. Metode `onCreate()` adalah yang paling penting untuk pemula; ini adalah tempat di mana semua inisialisasi satu kali untuk `Activity` Anda dilakukan, seperti mengatur tata letak (layout) UI.

Di dalam metode `onCreate()`, Anda akan memanggil `setContentView(R.layout.nama_file_layout_anda)`. Ini memberitahu `Activity` Anda file XML mana yang harus digunakan sebagai tata letak visual untuk layar ini. Setelah `Activity` dibuat dan tata letak diatur, `Activity` akan terlihat oleh pengguna dan siap untuk interaksi. Memahami `Activity` sangat penting karena ini adalah titik masuk utama bagi sebagian besar interaksi pengguna dan mengelola tampilan layar aplikasi Anda secara keseluruhan.

```
package com.example.myfirstapp

import android.os.Bundle
import androidx.appcompat.app.AppCompatActivity

class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main) // Menghubungkan file layout XML
    }
}
```

💡 **Fun Fact:** Tahukah Anda, Activity adalah salah satu komponen inti Android yang paling tua, sudah ada sejak versi Android 1.0! Ini menunjukkan betapa fundamentalnya peran Activity dalam setiap aplikasi Android.
