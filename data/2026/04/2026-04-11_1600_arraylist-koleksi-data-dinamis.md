## ☕ Java Concept: ArrayList: Koleksi Data Dinamis

**Category:** Collections
**TL;DR:** `ArrayList` adalah kelas di Java yang memungkinkan kita menyimpan daftar objek dengan ukuran yang bisa berubah secara dinamis, sangat sering dipakai di aplikasi nyata.

Sebagai mahasiswa semester awal, Anda mungkin sudah familiar dengan konsep array yang memiliki ukuran tetap. Namun, dalam banyak skenario pemrograman, kita seringkali membutuhkan daftar data yang ukurannya tidak kita ketahui secara pasti di awal, atau yang bisa bertambah dan berkurang selama program berjalan. Di sinilah `ArrayList` menjadi solusi yang sangat powerful dan sering digunakan. Ia merupakan bagian dari Java Collections Framework, sebuah set antarmuka dan kelas untuk merepresentasikan dan memanipulasi koleksi objek.

`ArrayList` memungkinkan Anda untuk membuat daftar objek tanpa perlu menentukan kapasitas awal. Anda dapat dengan mudah menambahkan elemen baru menggunakan metode `add()`, mengambil elemen pada indeks tertentu menggunakan `get()`, atau bahkan menghapus elemen menggunakan `remove()`. Hal ini sangat praktis jika Anda sedang memproses data seperti daftar nama pelanggan, daftar item di keranjang belanja, atau hasil query dari database yang jumlahnya bisa bervariasi. Kemudahan manipulasi dan sifatnya yang dinamis membuat `ArrayList` menjadi pilihan utama bagi banyak *developer*.

Saat membuat `ArrayList`, Anda bisa menggunakan *generics* (misalnya `<String>` atau `<Integer>`) untuk menentukan tipe data objek yang akan disimpan, sehingga program Anda menjadi lebih aman dan terhindar dari kesalahan tipe data saat kompilasi. Dengan memahami `ArrayList`, Anda telah mengambil langkah penting dalam menguasai pengelolaan data yang lebih efisien dan fleksibel dalam pemrograman Java, membuka pintu untuk memahami struktur data yang lebih kompleks di masa depan.

```
import java.util.ArrayList;

public class BelajarArrayList {
    public static void main(String[] args) {
        // Membuat ArrayList untuk menyimpan nama mahasiswa (tipe String)
        ArrayList<String> daftarMahasiswa = new ArrayList<>();
        
        // Menambahkan elemen ke ArrayList
        daftarMahasiswa.add("Budi");
        daftarMahasiswa.add("Ani");
        daftarMahasiswa.add("Cici");
        
        // Menampilkan seluruh daftar
        System.out.println("Daftar Mahasiswa: " + daftarMahasiswa);
        
        // Mengakses elemen pertama (indeks 0)
        System.out.println("Mahasiswa pertama: " + daftarMahasiswa.get(0));
        
        // Menghapus elemen "Ani"
        daftarMahasiswa.remove("Ani");
        
        // Menampilkan daftar setelah penghapusan
        System.out.println("Daftar setelah Ani dihapus: " + daftarMahasiswa);
    }
}
```

💡 **Fun Fact:** Meskipun `ArrayList` terlihat 'ajaib' karena ukurannya bisa berubah dinamis, di balik layarnya, ia sebenarnya masih menggunakan array biasa (regular array) sebagai tempat penyimpanan data. Ketika kapasitas array internalnya penuh, `ArrayList` akan secara otomatis membuat array baru yang lebih besar, menyalin semua elemen lama ke array baru tersebut, lalu membuang array yang lama. Proses ini terjadi secara transparan dan efisien untuk Anda!
