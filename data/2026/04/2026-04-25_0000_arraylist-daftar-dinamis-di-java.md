## ☕ Java Concept: ArrayList: Daftar Dinamis di Java

**Category:** Collections
**TL;DR:** ArrayList adalah implementasi dari antarmuka List yang menyediakan struktur data array yang ukurannya dapat berubah secara dinamis, sangat berguna untuk menyimpan koleksi objek.

Saat belajar Java, kita sering menggunakan array untuk menyimpan banyak data. Namun, array standar di Java memiliki ukuran tetap yang harus ditentukan di awal. Ini sering menjadi masalah jika kita tidak tahu berapa banyak data yang akan disimpan atau jika jumlah data akan sering berubah. Di sinilah `ArrayList` datang sebagai solusi.

`ArrayList` adalah bagian dari Java Collections Framework, yang memungkinkan kita untuk menyimpan koleksi objek dengan ukuran yang bisa membesar atau mengecil secara otomatis sesuai kebutuhan. Ini berarti kita tidak perlu khawatir tentang kehabisan ruang atau membuang memori yang tidak terpakai, karena `ArrayList` akan mengelola kapasitasnya sendiri di belakang layar.

Untuk mulai menggunakan `ArrayList`, kita perlu mengimpornya dari package `java.util`. Kita mendefinisikannya dengan menentukan tipe objek yang akan disimpan di dalamnya menggunakan generics, misalnya `ArrayList<String>` untuk menyimpan teks atau `ArrayList<Integer>` untuk menyimpan angka. Setelah dibuat, kita bisa menambah elemen dengan method `add()`, mengakses elemen dengan `get(index)`, atau menghapus elemen dengan `remove(index)` atau `remove(object)`.

Keunggulan utama `ArrayList` adalah kemampuannya untuk beradaptasi dengan jumlah data yang tidak pasti. Ini sangat memudahkan manajemen data dinamis seperti daftar siswa, item belanja, atau log peristiwa. Kita juga bisa dengan mudah mengulang (iterate) semua elemen di dalamnya menggunakan loop `for-each` yang sederhana, menjadikannya pilihan yang sangat fleksibel dan efisien untuk banyak skenario pemrograman.

```
import java.util.ArrayList;

public class DaftarBuku {
    public static void main(String[] args) {
        ArrayList<String> judulBuku = new ArrayList<>();
        judulBuku.add("Filosofi Teras");
        judulBuku.add("Atomic Habits");
        judulBuku.add("The Psychology of Money");

        System.out.println("Jumlah buku: " + judulBuku.size());
        System.out.println("Buku pertama: " + judulBuku.get(0));

        for (String judul : judulBuku) {
            System.out.println("Judul: " + judul);
        }
    }
}
```

💡 **Fun Fact:** Java Collections Framework, termasuk `ArrayList`, dirancang oleh seorang ilmuwan komputer bernama Joshua Bloch. Ia juga dikenal sebagai penulis buku "Effective Java," yang sangat direkomendasikan untuk setiap pengembang Java yang ingin menulis kode berkualitas tinggi.
