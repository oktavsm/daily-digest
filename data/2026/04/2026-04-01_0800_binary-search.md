## ⚡ Competitive Programming Concept: Binary Search

**Category:** Searching
**TL;DR:** Binary Search adalah algoritma pencarian yang sangat efisien untuk menemukan elemen dalam array yang terurut atau untuk mencari nilai optimal pada masalah dengan properti monoton.

Binary Search adalah algoritma pencarian fundamental yang memanfaatkan properti array yang terurut (ascending atau descending). Ide utamanya adalah "divide and conquer": kita membandingkan nilai target dengan elemen di tengah array, lalu menghilangkan setengah bagian array yang tidak mungkin mengandung target. Proses ini diulang sampai target ditemukan atau ruang pencarian habis.

Algoritma bekerja dengan menjaga dua pointer, `low` dan `high`, yang menandai batas-batas ruang pencarian saat ini. Pada setiap iterasi, kita menghitung `mid = low + (high - low) / 2`. Jika `arr[mid]` adalah target, kita selesai. Jika `arr[mid]` lebih kecil dari target, kita tahu target (jika ada) pasti berada di sebelah kanan `mid`, sehingga kita mengupdate `low = mid + 1`. Sebaliknya, jika `arr[mid]` lebih besar dari target, kita update `high = mid - 1`.

Selain mencari elemen di array terurut, Binary Search juga sangat kuat untuk mencari "jawaban" pada masalah di mana fungsi keputusan memiliki properti monoton (misalnya, jika `x` memenuhi kondisi, maka semua `y > x` juga memenuhi, atau sebaliknya). Ini sering disebut "binary search on answer". Kita melakukan binary search pada rentang nilai yang mungkin untuk jawaban, dan pada setiap `mid`, kita memeriksa apakah `mid` adalah jawaban yang valid menggunakan fungsi `check()` kustom.

Karena setiap langkah Binary Search mengurangi ruang pencarian menjadi setengahnya, kompleksitas waktunya sangat efisien, yaitu `O(log N)`, di mana `N` adalah ukuran array atau rentang pencarian. Ini jauh lebih cepat dibandingkan pencarian linear `O(N)` untuk ukuran `N` yang besar.

```
int binarySearch(const std::vector<int>& arr, int target) {
    int low = 0;
    int high = arr.size() - 1;
    while (low <= high) {
        int mid = low + (high - low) / 2;
        if (arr[mid] == target) {
            return mid; // Element found
        } else if (arr[mid] < target) {
            low = mid + 1;
        } else {
            high = mid - 1;
        }
    }
    return -1; // Element not found
}
```

💡 **Fun Fact:** Binary Search bisa digunakan untuk mencari akar kuadrat dari suatu bilangan dengan presisi tertentu, atau bahkan menemukan titik terendah pada fungsi cekung atau titik tertinggi pada fungsi cembung!
