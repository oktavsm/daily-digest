## ☕ Java Concept: Menggunakan ArrayList untuk Koleksi Dinamis

**Category:** Collections
**TL;DR:** ArrayList adalah implementasi List yang fleksibel, memungkinkan kita menyimpan urutan objek dan mengubah ukurannya secara dinamis.

Di Java, saat kita perlu menyimpan banyak data dengan tipe yang sama, kita sering menggunakan *array*. Namun, array memiliki batasan utama yaitu ukurannya sudah ditentukan dan tetap setelah dibuat. Ini menjadi masalah jika kita tidak tahu pasti berapa banyak data yang akan disimpan, atau jika jumlahnya bisa berubah-ubah selama program berjalan. Di sinilah `ArrayList` hadir sebagai solusi yang elegan dari Java Collections Framework.

`ArrayList` adalah salah satu implementasi dari antarmuka `List`, yang memungkinkan kita menyimpan urutan objek. Berbeda dengan array, ukuran `ArrayList` dapat bertambah atau berkurang secara otomatis (dinamis) sesuai kebutuhan. Ini berarti Anda tidak perlu khawatir tentang kapasitas awal; `ArrayList` akan menyesuaikan diri saat Anda menambah atau menghapus elemen. Untuk menggunakannya, Anda perlu mengimpor kelas `ArrayList` dari package `java.util`.

Untuk mulai menggunakan `ArrayList`, Anda mendeklarasikannya dengan menentukan tipe objek yang akan disimpan di dalamnya menggunakan *generics*, misalnya `ArrayList<String>` untuk menyimpan teks. Anda bisa menambahkan elemen menggunakan metode `add()`, mengakses elemen berdasarkan indeks (dimulai dari 0) menggunakan `get(int index)`, dan menghapus elemen dengan `remove(Object o)` atau `remove(int index)`. Anda juga bisa mengetahui berapa banyak elemen yang ada di `ArrayList` kapan saja menggunakan metode `size()`.

Keunggulan `ArrayList` adalah kemudahannya dan fleksibilitasnya dalam manajemen data. Ini sangat praktis untuk aplikasi di mana jumlah data tidak tetap, seperti daftar belanjaan yang bisa bertambah atau berkurang, daftar siswa yang mendaftar atau keluar, atau log peristiwa. Karena kepraktisannya, `ArrayList` adalah salah satu struktur data koleksi yang paling sering digunakan dalam pengembangan aplikasi Java sehari-hari.

```
import java.util.ArrayList;
import java.util.List;

public class ContohArrayList {
    public static void main(String[] args) {
        List<String> daftarNamaMahasiswa = new ArrayList<>(); // Deklarasi dan inisialisasi
        daftarNamaMahasiswa.add("Budi"); // Menambah elemen
        daftarNamaMahasiswa.add("Citra");
        daftarNamaMahasiswa.add("Dian");

        System.out.println("Mahasiswa pertama: " + daftarNamaMahasiswa.get(0)); // Mengakses elemen
        daftarNamaMahasiswa.remove("Citra"); // Menghapus elemen
        System.out.println("Jumlah mahasiswa sekarang: " + daftarNamaMahasiswa.size()); // Ukuran koleksi
    }
}
```

💡 **Fun Fact:** Meskipun `ArrayList` sangat populer, ada juga `LinkedList` yang bekerja berbeda untuk kasus penggunaan tertentu, menunjukkan kekayaan pilihan struktur data di Java Collections Framework.
