## ☕ Java Concept: ArrayList: Koleksi Dinamis Praktis

**Category:** Collections
**TL;DR:** ArrayList adalah implementasi dinamis dari antarmuka List di Java, memungkinkan kita menyimpan urutan objek yang ukurannya bisa berubah secara otomatis seiring waktu.

Pada pemrograman, seringkali kita perlu menyimpan sejumlah objek, tetapi kita tidak tahu persis berapa banyak objek yang akan disimpan di awal. Array biasa di Java memiliki ukuran yang tetap setelah dibuat, yang bisa menjadi batasan. Di sinilah `ArrayList` hadir sebagai solusi yang elegan dan sangat sering digunakan. `ArrayList` adalah bagian dari Java Collections Framework, yang menyediakan cara-cara terstruktur untuk mengelola kumpulan objek.

`ArrayList` bekerja seperti array, yaitu menyimpan elemen berdasarkan indeks (dimulai dari 0), namun dengan keunggulan utama: ukurannya tidak tetap. Kita bisa menambahkan (`add()`) atau menghapus (`remove()`) elemen kapan saja, dan `ArrayList` akan secara otomatis menyesuaikan kapasitasnya. Ini berarti kita tidak perlu khawatir tentang 'array penuh' seperti pada array konvensional. Untuk mengambil elemen, kita menggunakan metode `get(indeks)`, dan untuk mengetahui jumlah elemen, kita gunakan `size()`.

Penggunaan `ArrayList` juga sangat memudahkan dalam hal keamanan tipe data berkat fitur Generics. Saat membuat `ArrayList`, kita bisa menentukan tipe objek apa yang akan disimpannya (misalnya, `ArrayList<String>` untuk menyimpan String, atau `ArrayList<Integer>` untuk Integer). Ini membantu mencegah kesalahan tipe data saat kompilasi, membuat kode lebih robust dan mudah dibaca. `ArrayList` sangat praktis untuk berbagai skenario, seperti daftar nama pengguna, daftar item belanja, atau catatan nilai mahasiswa yang jumlahnya bisa bertambah atau berkurang.

```
import java.util.ArrayList;
import java.util.List;

public class ContohArrayList {
    public static void main(String[] args) {
        // Membuat ArrayList untuk menyimpan String
        List<String> daftarNama = new ArrayList<>();

        daftarNama.add("Budi");
        daftarNama.add("Siti");
        daftarNama.add("Andi");

        System.out.println("Nama pertama: " + daftarNama.get(0)); // Output: Budi
        System.out.println("Jumlah nama: " + daftarNama.size());   // Output: 3

        daftarNama.remove(1); // Menghapus "Siti"
        System.out.println("Daftar setelah dihapus: " + daftarNama); // Output: [Budi, Andi]
    }
}
```

💡 **Fun Fact:** Tahukah kamu, `ArrayList` sebenarnya dibangun di atas array biasa di balik layarnya? Ketika kamu menambahkan terlalu banyak elemen sehingga array internalnya penuh, `ArrayList` secara otomatis akan membuat array baru yang lebih besar (biasanya 1.5 kali ukuran semula) dan menyalin semua elemen lama ke array baru tersebut!
