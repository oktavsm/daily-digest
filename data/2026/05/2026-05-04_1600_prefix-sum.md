## ⚡ Competitive Programming Concept: Prefix Sum

**Category:** Data Structure
**TL;DR:** Teknik prekomputasi untuk menghitung jumlah elemen dalam rentang (range sum) dari sebuah array secara efisien dengan waktu O(1) per query setelah prekomputasi O(N).

Prefix Sum adalah teknik fundamental yang digunakan untuk mengoptimalkan perhitungan jumlah elemen dalam sub-array (range sum). Konsep dasarnya adalah membuat sebuah array baru, yang disebut array prefix sum, di mana setiap elemen `prefix_sum[i]` menyimpan total semua elemen dari indeks awal hingga indeks `i-1` dari array asli.

Untuk membangun array `prefix_sum`, kita melakukan iterasi tunggal melalui array asli. Jika array asli adalah `arr` dengan ukuran `N`, kita dapat membuat `prefix_sum` berukuran `N+1`. `prefix_sum[0]` biasanya 0. Kemudian, untuk setiap `i` dari 0 hingga `N-1`, kita set `prefix_sum[i+1] = prefix_sum[i] + arr[i]`. Proses prekomputasi ini membutuhkan kompleksitas waktu O(N), di mana N adalah jumlah elemen dalam array asli.

Setelah array `prefix_sum` terbentuk, kita dapat menjawab pertanyaan tentang jumlah elemen dalam rentang `[L, R]` (inklusi, 0-indexed) dengan sangat cepat. Jumlah elemen dari indeks `L` hingga `R` dapat dihitung sebagai `prefix_sum[R+1] - prefix_sum[L]`. Setiap query rentang dapat diselesaikan dalam waktu O(1), jauh lebih cepat daripada O(N) jika dihitung secara langsung.

Teknik ini sangat serbaguna dan sering muncul dalam berbagai masalah, baik secara langsung maupun sebagai bagian dari solusi yang lebih kompleks. Ia dapat diperluas ke dua dimensi (prefix sum 2D) untuk menghitung jumlah sub-matriks, dan menjadi dasar untuk memahami struktur data yang lebih maju seperti Fenwick Tree (BIT) atau Segment Tree.

```
vector<int> arr = {1, 2, 3, 4, 5};
vector<int> prefix_sum(arr.size() + 1, 0);

// Prekomputasi array prefix sum
for (int i = 0; i < arr.size(); ++i) {
    prefix_sum[i+1] = prefix_sum[i] + arr[i];
}

// Contoh query: hitung sum dari index 1 hingga 3 (nilai 2+3+4)
int L = 1; // Start index (0-indexed)
int R = 3; // End index (0-indexed)
int range_sum = prefix_sum[R+1] - prefix_sum[L]; // Hasilnya 9
```

💡 **Fun Fact:** Meskipun sangat sederhana, konsep prefix sum adalah salah satu contoh paling awal dan paling elegan dari 'prekomputasi' dalam ilmu komputer, sebuah strategi umum untuk mempercepat respons terhadap query dengan mengorbankan waktu setup awal.
