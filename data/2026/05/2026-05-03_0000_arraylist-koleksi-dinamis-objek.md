## ☕ Java Concept: ArrayList: Koleksi Dinamis Objek

**Category:** Collections
**TL;DR:** ArrayList adalah implementasi dari antarmuka List yang memungkinkan kita menyimpan koleksi objek dengan ukuran yang bisa berubah secara dinamis.

Selamat pagi, mahasiswa! Hari ini kita akan menyelami salah satu struktur data paling fundamental dan sering digunakan di Java, yaitu ArrayList. Sampai saat ini, mungkin kalian sudah terbiasa dengan array biasa yang ukurannya sudah ditentukan di awal dan tidak bisa diubah. Nah, di dunia nyata, seringkali kita tidak tahu berapa banyak data yang akan kita simpan. Di sinilah ArrayList datang sebagai pahlawan, memberikan kita fleksibilitas untuk memiliki 'array' yang bisa membesar atau mengecil sesuai kebutuhan.

ArrayList merupakan bagian dari Java Collections Framework dan mengimplementasikan interface List. Ini berarti ia mempertahankan urutan elemen yang ditambahkan (insertion order) dan memungkinkan adanya duplikasi elemen. Untuk menggunakan ArrayList, kita perlu mengimpornya dari paket `java.util`. Pembuatannya sangat mudah: cukup tentukan tipe data objek yang ingin disimpan di dalamnya menggunakan Generics (misalnya, `ArrayList<String>`), lalu instansiasikan objek ArrayList baru.

Beberapa operasi dasar yang akan sering kalian gunakan adalah menambahkan elemen dengan `add()`, mengakses elemen berdasarkan indeks dengan `get(index)`, dan mendapatkan jumlah elemen yang ada saat ini dengan `size()`. Keunggulan utama ArrayList adalah kemampuannya untuk secara otomatis mengatur kapasitas penyimpanannya. Ketika ArrayList penuh dan kita mencoba menambahkan elemen baru, ia akan secara otomatis membuat array internal yang lebih besar dan menyalin semua elemen lama ke array baru tersebut. Proses ini transparan bagi kita sebagai programmer, membuat pengembangan lebih efisien.

Selain itu, ArrayList juga menyediakan berbagai metode lain yang sangat berguna, seperti `remove(index)` atau `remove(object)` untuk menghapus elemen, `set(index, element)` untuk mengubah elemen pada indeks tertentu, dan `clear()` untuk mengosongkan seluruh ArrayList. Dengan ArrayList, kalian bisa dengan mudah mengelola daftar data yang dinamis, mulai dari daftar nama siswa, produk di toko online, hingga riwayat transaksi, tanpa perlu khawatir tentang ukuran maksimal yang harus ditentukan di awal.

```
import java.util.ArrayList;

public class DaftarMahasiswa {
    public static void main(String[] args) {
        ArrayList<String> daftarNama = new ArrayList<>();
        daftarNama.add("Budi");
        daftarNama.add("Siti");
        daftarNama.add("Andi");

        System.out.println("Daftar Mahasiswa: " + daftarNama);
        System.out.println("Mahasiswa ke-2: " + daftarNama.get(1));
        System.out.println("Jumlah mahasiswa: " + daftarNama.size());
    }
}
```

💡 **Fun Fact:** Meskipun ArrayList terlihat seperti array biasa yang fleksibel, di balik layarnya ArrayList sebenarnya masih menggunakan array sebagai pondasi penyimpanannya! Ketika kalian menambahkan elemen sehingga ArrayList melebihi kapasitasnya, ia akan membuat array internal baru yang lebih besar (biasanya 1.5 kali ukuran semula) dan menyalin semua elemen dari array lama ke array baru. Proses ini disebut 'resizing' atau 'reallocation'.
