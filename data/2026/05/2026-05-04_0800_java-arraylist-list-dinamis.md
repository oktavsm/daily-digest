## ☕ Java Concept: Java ArrayList: List Dinamis

**Category:** Collections
**TL;DR:** ArrayList adalah implementasi antarmuka List yang sangat sering digunakan untuk menyimpan dan mengelola sekumpulan objek secara dinamis, artinya ukurannya bisa berubah otomatis.

Dalam pemrograman, seringkali kita perlu menyimpan sekumpulan data yang jumlahnya tidak tetap. Array biasa memiliki ukuran tetap, yang bisa menyulitkan jika kita sering menambah atau menghapus data. Di sinilah `ArrayList` hadir sebagai solusi. `ArrayList` adalah salah satu kelas fundamental dalam Java Collections Framework yang menyediakan fungsionalitas 'list' atau daftar objek yang ukurannya bisa membesar dan mengecil secara otomatis sesuai kebutuhan.

Keunggulan utama `ArrayList` adalah kemudahannya dalam manajemen data. Kamu tidak perlu khawatir tentang kapasitas maksimal seperti pada array tradisional. Kamu bisa dengan mudah menambahkan elemen baru menggunakan metode `add()`, mengambil elemen di posisi tertentu dengan `get()`, atau menghapus elemen menggunakan `remove()`. `ArrayList` akan secara internal menangani semua detail perubahan ukuran, memberikan abstraksi yang bersih bagi programmer.

`ArrayList` sangat praktis untuk berbagai skenario, mulai dari menyimpan daftar nama mahasiswa, daftar item belanja, hingga log peristiwa. Karena sifatnya yang dinamis, `ArrayList` menjadi pilihan yang tepat ketika jumlah data yang akan kamu olah tidak diketahui secara pasti di awal program atau sering berubah selama eksekusi. Ini membuat kode lebih fleksibel dan mudah dipelihara.

Meskipun praktis, penting untuk diingat bahwa `ArrayList` paling efisien untuk operasi menambah dan mengakses elemen di akhir daftar, serta mengakses elemen berdasarkan indeks. Operasi menghapus atau menambah elemen di tengah daftar bisa sedikit kurang efisien karena memerlukan pergeseran banyak elemen lainnya, namun untuk sebagian besar kasus penggunaan, performanya sudah sangat memadai.

```
import java.util.ArrayList;

ArrayList<String> daftarTugas = new ArrayList<>(); // Deklarasi dan inisialisasi
daftarTugas.add("Belajar Java");        // Menambah elemen
daftarTugas.add("Kerjakan PR");
daftarTugas.add("Olahraga");

String tugasPertama = daftarTugas.get(0); // Mengambil elemen di indeks 0
System.out.println("Tugas pertama: " + tugasPertama); // Output: Belajar Java

daftarTugas.remove(1); // Menghapus elemen di indeks 1 ("Kerjakan PR")
System.out.println("Jumlah tugas tersisa: " + daftarTugas.size()); // Output: 2
```

💡 **Fun Fact:** Tahukah kamu, di balik layar, `ArrayList` sebenarnya menggunakan array biasa? Saat `ArrayList` penuh dan kamu ingin menambah elemen baru, ia akan membuat array baru yang lebih besar (biasanya 1.5 kali ukuran semula), menyalin semua elemen lama ke array baru itu, lalu membuang array yang lama. Inilah yang membuatnya 'dinamis'!
